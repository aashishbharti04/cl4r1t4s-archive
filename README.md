# CL4R1T4S archive + analysis

A personal archive of the [`elder-plinius/CL4R1T4S`](https://github.com/elder-plinius/CL4R1T4S)
collection of extracted/claimed LLM system prompts, plus original comparison analysis.

## Contents

- Vendor folders (`ANTHROPIC/`, `OPENAI/`, `GOOGLE/`, `XAI/`, `PERPLEXITY/`, coding tools, etc.) —
  mirrored from the source repo at commit `09916a9` (2026-06-15).
- [`diff-analysis/`](diff-analysis/) — original analysis written for this archive:
  - `fable5-vs-opus47.md` — generational diff of the Claude Opus 4.7 → Claude Fable 5 system prompts.

The prompt files are not my work. They originate from the public `elder-plinius/CL4R1T4S`

contain paraphrased, reordered, or hallucinated passages and are not guaranteed to be verbatim.
All credit for the source collection goes to the original author. This repo exists as a personal
archive and as a home for the comparison notes in `diff-analysis/`.
